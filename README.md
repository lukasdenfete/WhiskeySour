WHISKEY SOUR CARPARTS
ABOUT
An e-commerce platform for vintage car parts, featuring a community section where users can connect through a chat forum and private messages. The application follows a server-side rendering approach with a layered architecture.

FEATURES
E-commerce

Product Catalog: Browse products by category, view detailed product pages, and see related items.

Shopping Cart: Database-backed shopping cart functionality allowing users to add, update, and remove items (persists across sessions).

Search: Integrated search functionality for products and users.

Community

Forum: Create threads, post comments, and like content.

Messaging: Private messaging system between users.

Social: Follow other users and view their profiles.

Notifications: System for unread messages and interactions.

Core & Security

Authentication: Secure user registration and login using ASP.NET Identity.

Authorization: Role-based access control (Admin/User).

Data Handling: Image upload support and dynamic stock management.
